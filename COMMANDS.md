# 📱 قائمة الأوامر BLE - التفاصيل الكاملة

## الأوامر الأساسية

### 1️⃣ التشغيل والإطفاء

#### أمر التشغيل
```
ON
```
- **الوصف**: تشغيل جميع الليدات الـ 8
- **المتوقع**: الليدات تضيء باللون والسطوع الحالي
- **مثال الاستخدام**:
  ```
  اكتب في التطبيق: ON
  ثم اضغط: Send
  ```

#### أمر الإطفاء
```
OFF
```
- **الوصف**: إطفاء جميع الليدات
- **المتوقع**: كل الليدات تطفأ (أسود)
- **مثال الاستخدام**:
  ```
  اكتب في التطبيق: OFF
  ثم اضغط: Send
  ```

---

### 2️⃣ التحكم بالسطوع (Brightness)

#### زيادة السطوع
```
BRIGHTNESS_UP
```
- **الوصف**: زيادة السطوع بمقدار **13 نقطة** (≈ 5% من 255)
- **النطاق**: من 0 (مظلم جداً) إلى 255 (ساطع جداً)
- **الحد الأقصى**: 255
- **الحد الأدنى**: 0
- **مثال الاستخدام**:
  ```
  اكتب: BRIGHTNESS_UP
  كرر عدة مرات للحصول على سطوع أقوى
  ```

#### خفض السطوع
```
BRIGHTNESS_DOWN
```
- **الوصف**: خفض السطوع بمقدار **13 نقطة** (≈ 5% من 255)
- **النطاق**: من 255 (ساطع جداً) إلى 0 (مظلم جداً)
- **مثال الاستخدام**:
  ```
  اكتب: BRIGHTNESS_DOWN
  كرر عدة مرات للحصول على ضوء خافت
  ```

**جدول السطوع:**
| العدد | النسبة | الوصف |
|------|--------|-------|
| 255 | 100% | ساطع جداً |
| 204 | 80% | ساطع |
| 153 | 60% | متوسط |
| 102 | 40% | خافت |
| 51 | 20% | خافت جداً |
| 0 | 0% | مظلم (مطفأ) |

---

### 3️⃣ التحكم بدرجة اللون (Color Hue)

#### رفع درجة اللون
```
COLOR_UP
```
- **الوصف**: رفع درجة اللون بمقدار **5 درجات** في دائرة الألوان
- **النطاق**: 0-255 (دائرة ألوان كاملة)
- **الحركة**: من الأحمر → برتقالي → أصفر → أخضر → أزرق → بنفسجي → أحمر

#### خفض درجة اللون
```
COLOR_DOWN
```
- **الوصف**: خفض درجة اللون بمقدار **5 درجات** في دائرة الألوان
- **الحركة**: معاكس (بنفسجي → أزرق → أخضر → أصفر → برتقالي → أحمر)

**دائرة الألوان الكاملة:**
```
القيمة (Hue)    الوصف        اللون
─────────────────────────────────
0               أحمر         🔴
42              برتقالي      🟠
85              أصفر         🟡
127             أخضر         🟢
170             أزرق         🔵
212             بنفسجي       🟣
255             أحمر (يعود)   🔴
```

**مثال الاستخدام:**
```
الحالة الحالية: أحمر (0)

1. أرسل: COLOR_UP
   النتيجة: برتقالي (5)

2. أرسل: COLOR_UP
   النتيجة: أصفر-برتقالي (10)

3. أرسل: COLOR_UP (x17 مرة إضافية)
   النتيجة: أصفر (85)
```

### 4️⃣ المودات الخاصة (Special Modes)

#### مود نبضات الألوان 💫 (Pulse Mode)
```
MODE_PULSE
```

**المميزات:**
- الليدات تنبض بسلاسة (تفتح وتغلق)
- السطوع يتغير من **50** إلى **255** تلقائياً
- اللون يبقى ثابتاً (آخر لون محدد)
- تأثير هادئ وجميل
- مثالي للأجواء الهادئة

**سرعة النبض:**
- كل 50 ميلي ثانية: تحديث السطوع
- دورة كاملة (نبضة واحدة): ~2.5 ثانية

**مثال الاستخدام:**
```
1. أرسل: ON
2. أرسل: COLOR_UP (لاختيار لون)
3. أرسل: MODE_PULSE
→ الليدات تبدأ تنبض باللون المختار
```

---

#### مود تأثير النار 🔥 (Fire Mode)
```
MODE_FIRE
```

**المميزات:**
- محاكاة حقيقية لتأثير اللهب
- ألوان عشوائية: أحمر 🔴 برتقالي 🟠 أصفر 🟡
- ارتعاش عشوائي يحاكي حركة النار الطبيعية
- كل LED يتغير بشكل مستقل
- تأثير ديناميكي وحي جداً

**آلية العمل:**
- كل LED يختار لون عشوائي من (أحمر - برتقالي - أصفر)
- يطبق ارتعاش عشوائي (0-255) على كل لون
- التحديث كل 100 ميلي ثانية

**مثال الاستخدام:**
```
1. أرسل: ON
2. أرسل: MODE_FIRE
→ تأثير النار يبدأ فوراً
→ الليدات تتحرك وترتعش مثل النار الحقيقية
```

**التأثير البصري:**
```
الوقت 1:        الوقت 2:        الوقت 3:
🔴 🟠 🟡        🟡 🔴 🟠        🟠 🟡 🔴
🟠 🟡 🔴        🟠 🟡 🔴        🟡 🔴 🟠
(يتكرر بشكل عشوائي)
```

## سيناريوهات الاستخدام العملية

### سيناريو 1: إضاءة بسيطة ثابتة
```
1. أرسل: ON
2. أرسل: BRIGHTNESS_UP (2-3 مرات للسطوع المناسب)
3. أرسل: COLOR_UP (عدة مرات للون المفضل)
↓
نتيجة: إضاءة ثابتة بلون وسطوع محددين
```

### سيناريو 2: أجواء هادئة ومريحة
```
1. أرسل: ON
2. أرسل: BRIGHTNESS_DOWN (لخفض السطوع)
3. أرسل: COLOR_UP (لاختيار لون أزرق أو أخضر)
4. أرسل: MODE_PULSE
↓
نتيجة: نبضات هادئة في ضوء خافت (رومانسي)
```

### سيناريو 3: حفلة أو حدث خاص
```
1. أرسل: ON
2. أرسل: BRIGHTNESS_UP (عدة مرات)
3. أرسل: MODE_PULSE أو MODE_FIRE
↓
نتيجة: تأثير ديناميكي جميل (احتفالي)
```

### سيناريو 4: إطفاء سريع
```
1. أرسل: OFF
↓
نتيجة: جميع الليدات تطفأ فوراً
```

## ملاحظات مهمة ⚠️

### 1. التتابع الصحيح
```
✅ صحيح:
- ON → BRIGHTNESS_UP → COLOR_UP → MODE_PULSE

❌ خطأ:
- BRIGHTNESS_UP (بدون ON أولاً)
- COLOR_UP (بدون تشغيل)
```

### 2. أسماء الأوامر (Case Sensitive)
```
✅ صحيح:
- ON (أحرف كبيرة)
- BRIGHTNESS_UP
- MODE_FIRE

❌ خطأ:
- on (أحرف صغيرة)
- On (مختلط)
- brightness_up (أحرف صغيرة)
```

### 3. بدون مسافات إضافية
```
✅ صحيح:
ON

❌ خطأ:
ON  (مسافة في النهاية)
 ON (مسافة في البداية)
O N (مسافة في الوسط)
```

### 4. تغيير المود
```
- عندما تختار مود (MODE_PULSE أو MODE_FIRE)
- المود يستمر حتى تختار مود آخر
- أرسل ON/OFF لتعديل حالة الليدات بدون تغيير المود
```

## استكشاف الأخطاء 🔧

| المشكلة | الحل |
|--------|------|
| أمر غير معروف | تحقق من كتابة الأمر بشكل صحيح (أحرف كبيرة) |
| الليدات لا تستجيب | أعد الاتصال بالبلوتوث |
| السطوع لا يتغير | تأكد من أن الليدات مشغلة (ON) |
| اللون لا يتغير | جرب COLOR_UP عدة مرات للوصول للون المراد |
| المود لا يعمل | تأكد من أن الليدات مشغلة قبل تفعيل المود |
